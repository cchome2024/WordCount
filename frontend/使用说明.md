# 使用说明

## 怎么用

### 1. 上传文件
- **电脑上**：点击上传区域选择文件，或直接把文件拖到上传区域
- **iPhone上**：点击上传区域，然后：
  - 选择"浏览"或"文件"
  - 可以从"文件"app选择
  - 可以从iCloud Drive选择
  - 可以从其他app分享的文件中选择
- 可以一次上传多个文件

### 2. 等待处理
- 文件上传后会自动处理
- 看到"已完成"就可以看结果了

### 3. 查看结果
会显示两个表格：
- **第一个表格**：单词表中的单词统计（677个单词）
- **第二个表格**：其他单词统计（排除日常用词）

### 4. 复制数据
每个表格右上角有两个按钮：
- **复制完整**：复制排名、单词、次数（可以粘贴到Excel）
- **复制单词**：只复制单词列表

### 5. 过滤功能（可选）
在第二个表格的右上角，可以选择"最小长度"：
- 比如选择"5个字母及以上"，就只显示5个字母以上的单词

### 6. 重新开始
- 点击页面右上角的"Reset Analysis"按钮
- 会清空所有文件和结果，可以重新上传新文件

## iPhone 使用技巧

### 方法一：从"文件"app选择
1. 点击上传区域
2. 选择"浏览"或"文件"
3. 在文件app中找到你的Word或PDF文件
4. 选择文件即可

### 方法二：从微信直接传文件（最简单的方法）

**一次设置，以后都方便：**

1. **在微信中保存文件**（只需做一次）
   - 在微信中打开文件（PDF或Word）
   - 点击右上角"分享" → 选择"存储到文件"
   - 选择"我的iPhone" → 点击"存储"
   - 以后微信收到的文件都可以这样保存

2. **在网页中上传**（以后都这样操作）
   - 打开单词统计网页
   - 点击上传区域
   - 选择"浏览" → 选择"我的iPhone"
   - 找到文件并选择即可

**为什么需要先保存？**
- iPhone的安全机制：每个app的文件是隔离的
- 微信的文件不能直接传给网页，必须先保存到"文件"app
- 保存一次后，以后就可以直接选择了

**更快的操作：**
- 保存文件时选择"我的iPhone"（不要选iCloud Drive）
- 这样在网页中选择时，直接点"我的iPhone"就能看到

### 方法三：从iCloud Drive选择
1. 点击上传区域
2. 选择"浏览"
3. 选择"iCloud Drive"
4. 找到你的文件并选择

## 注意事项

- 只支持 Word (.docx) 和 PDF 文件（PDF解析可能有乱码，建议先转Word上传）
- 扫描版PDF（图片格式）无法处理
- 加密的PDF文件无法处理

## 简单三步

1. **上传文件** → 2. **看结果** → 3. **复制数据**

就这么简单！
